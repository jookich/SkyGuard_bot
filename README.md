# SkyGuard_bot

# Обзор файлов бота
1. bot.py - основной файл со структурой бота, для запуска использовать только его
2. config.py - файл с API-ключами для бота и сервиса по погоде\
3. safety_logic.py - файл с логикой проверки безопасности полета, все условия и сравнению прописаны здесь
4. weather_api.py - файл с функцией запроса к метеосервису для получения данных о погоде в виде JSON-файла


# Инструкция по использованию

1. Скачиваете файлы
2. Устанавливаете Python версии 3.12 или выше
3. Устанавливаете любое IDE(Microsoft Visual Code или PyCharm Community)
4. Устанавливаете библиотеку aiogram версии 3.26.0(через терминал IDE)
   Все остальные библиотеки нужной версии установятся вместе с установкой aiogram
5. Пробуете запускать, если что-то не запускается, пишите мне либо ищите в интернете как что-то исправить
